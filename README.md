## Logged Data
Each object records (in `.csv` file):
- Date & time
- Runtime
- Object name
- Object position (`x,y,z`)
- Object rotation (`rx,ry,rz`)
- Touch status
- Grab status
- Time to grab from touching
- Distance from touch to grab
- Grab events (`GrabStart, GrabEnd, TouchNoGrab`)
- Grab count
- Grab durations
- Hover without grab count
- Viewing side
- Viewing angles in dot products:
    - Front 
    - Right
    - Up

---
This project was developed for the Marketing Department at  Robert H. Smith School of Business to explore XR interaction analytics and real-world object integration in Unity.
